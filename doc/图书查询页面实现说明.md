# 图书查询页面实现说明

## 📋 功能概述

读者图书查询页面已完成开发，提供完整的图书检索和浏览功能。

## ✅ 已实现功能

### 1. 搜索栏
- **书名搜索**：支持模糊匹配
- **作者搜索**：支持模糊匹配
- **图书类型筛选**：下拉选择图书分类
- **搜索按钮**：执行查询操作
- **重置按钮**：清空所有搜索条件

### 2. 图书列表
- **表格展示**：清晰展示图书信息
  - 序号
  - 书名
  - 作者
  - 图书类型
  - 价格（红色高亮显示）
  - 状态（可借/已借出，标签显示）
  - 描述
  - 操作按钮

- **状态标识**：
  - 可借：绿色标签
  - 已借出：红色标签

- **借阅按钮**：
  - 可借图书：显示"借阅"按钮（蓝色）

### 3. 分页功能
- **分页器**：支持翻页浏览
- **每页数量**：可选择 10/20/50/100 条
- **总数显示**：显示图书总数
- **页码跳转**：快速跳转到指定页

### 4. 导航系统
- **顶部导航栏**：
  - 系统标题
  - 用户名显示
  - 角色标签（管理员/读者）
  - 退出登录按钮

- **侧边栏菜单**：
  - 首页
  - 图书查询 ✅
  - 借阅管理（开发中）
  - 修改密码（开发中）
  - 用户管理（管理员，开发中）
  - 类型管理（管理员，开发中）

### 5. 首页仪表盘
- **欢迎卡片**：渐变背景，显示欢迎信息
- **快捷入口**：图标式快捷访问
- **系统信息**：显示用户信息和系统版本

## 🔌 API 对接

### 图书查询接口
```
GET /bookInfo/queryBookInfosByPage
```

**请求参数：**
- `page`：页码（必填）
- `limit`：每页数量（必填）
- `bookname`：书名（可选）
- `bookauthor`：作者（可选）
- `booktypeid`：图书类型ID（可选）

**响应格式：**
```json
{
  "code": 0,
  "message": "success",
  "count": 200,
  "data": [
    {
      "bookid": 1,
      "bookname": "Java编程思想",
      "bookauthor": "Bruce Eckel",
      "bookprice": 108.00,
      "booktypeid": 1,
      "booktypename": "计算机科学",
      "bookdesc": "Java经典书籍",
      "isborrowed": 0,
      "bookimg": "/upload/book1.jpg"
    }
  ]
}
```

### 图书类型接口
```
GET /bookType/reader/queryBookTypes
```

**响应格式：**
```json
[
  {
    "booktypeid": 1,
    "booktypename": "计算机科学",
    "booktypedesc": "计算机相关书籍"
  }
]
```

## 🚀 使用说明

### 访问路径
```
http://localhost:5173/books
```

### 操作流程
1. 登录系统
2. 点击侧边栏"图书查询"菜单
3. 输入搜索条件（可选）
4. 点击"搜索"按钮
5. 浏览图书列表
6. 点击"借阅"按钮借书（功能开发中）

### 搜索示例
- **按书名搜索**：输入"Java"，查找所有包含"Java"的图书
- **按作者搜索**：输入"周志明"，查找该作者的所有图书
- **按类型筛选**：选择"计算机科学"，查找该类型的所有图书
- **组合搜索**：同时使用多个条件进行精确查找

## ⚠️ 注意事项

1. **后端服务**：确保后端运行在 `http://localhost:8092`
2. **Token 验证**：需要先登录获取 Token
3. **借阅功能**：借阅按钮已预留，功能待开发
4. **权限控制**：读者和管理员均可访问此页面

**数据加载：**
```javascript
const loadBookList = async () => {
  loading.value = true
  try {
    const params = {
      page: pagination.page,
      limit: pagination.limit,
      ...searchForm
    }
    const response = await queryBooksByPage(params)
    bookList.value = response.data || []
    total.value = response.count || 0
  } finally {
    loading.value = false
  }
}
```
