## 📝 API 对接说明

### 登录接口

**请求：**
```
POST http://localhost:8092/BookManager/user/login
Content-Type: application/json

{
  "username": "admin",
  "userpassword": "123456",
  "isadmin": 1
}
```

**响应：**
```json
{
  "status": 200,
  "message": "登录成功",
  "data": {
    "token": "a1b2c3d4e5f6..."
  },
  "timestamp": 1711180800
}
```

### Token 使用

登录成功后，token 会被保存到 localStorage，后续所有请求会自动携带 token 参数。

## ⚠️ 注意事项

1. **后端服务**：确保后端服务运行在 `http://localhost:8092`
2. **跨域问题**：如遇跨域，需要后端配置 CORS
3. **Token 过期**：Token 有效期 1 小时，过期需重新登录
4. **示例组件**：`src/components/` 下的示例组件可以删除
