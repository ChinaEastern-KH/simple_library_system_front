# 登录页面实现说明

## 已创建的文件

### 1. 核心配置文件

#### `src/utils/request.js`
Axios 请求封装，包含：
- 基础 URL 配置：`http://localhost:8092/BookManager`
- 请求拦截器：自动添加 token 参数
- 响应拦截器：统一处理成功/失败响应，自动显示错误提示

#### `src/api/user.js`
用户相关 API 接口：
- `login(data)` - 用户登录
- `getUserInfo(token)` - 获取用户信息
- `logout(token)` - 用户登出
- `register(data)` - 用户注册

#### `src/router/index.js`
路由配置：
- `/login` - 登录页
- `/` - 首页
- 路由守卫：未登录自动跳转到登录页

### 2. 页面组件

#### `src/views/Login.vue`
登录页面，功能包括：
- 用户名输入（2-20字符验证）
- 密码输入（6-20字符验证，支持显示/隐藏）
- 角色选择（读者/管理员）
- 表单验证
- 登录成功后保存 token 到 localStorage
- 跳转注册页入口

#### `src/views/Home.vue`
首页（临时占位页面），功能包括：
- 显示欢迎信息和用户角色
- 退出登录功能
- 顶部导航栏

### 3. 入口文件

#### `src/main.js`
应用入口，配置：
- 引入 Element Plus 及其样式
- 注册所有 Element Plus 图标
- 挂载 Vue Router

#### `src/App.vue`
根组件，使用 `<router-view />` 渲染路由页面

## 登录流程

```
1. 用户访问 / → 路由守卫检测无 token → 重定向到 /login
2. 用户填写表单（用户名、密码、角色）
3. 点击登录按钮 → 表单验证
4. 调用 POST /user/login 接口
5. 登录成功 → 保存 token、username、isadmin 到 localStorage
6. 跳转到首页 /
7. 首页显示欢迎信息
```

## 技术特点

### 1. 响应式设计
- 使用 Element Plus 组件库
- 渐变色背景 + 卡片式登录框
- 居中布局，适配各种屏幕

### 2. 表单验证
- 用户名：必填，2-20字符
- 密码：必填，6-20字符
- 角色：必选

### 3. 用户体验
- 密码可显示/隐藏
- 输入框支持清空
- 加载状态提示
- 错误消息自动提示（Element Plus Message）

### 4. 安全性
- Token 存储在 localStorage
- 路由守卫保护未授权访问
- 请求拦截器自动携带 token
