# 快速启动指南

## 前置条件

- Node.js 22.16 或更高版本
- npm 包管理器
- 后端服务运行在 `http://localhost:8092`

## 启动步骤

### 1. 安装依赖（首次运行）

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务将在 `http://localhost:5173` 启动。

### 3. 访问登录页

在浏览器中打开：`http://localhost:5173/login`

## 常见问题

### Q1: 登录时提示"网络请求失败"
**A:** 请确保后端服务已启动并运行在 `http://localhost:8092`

### Q2: 页面样式显示异常
**A:** 请确保已正确安装 Element Plus：
```bash
npm install element-plus
```

### Q3: 路由跳转不生效
**A:** 请确保已安装 Vue Router：
```bash
npm install vue-router@4
```

### Q4: 如何修改 API 地址
**A:** 编辑 `src/utils/request.js` 文件中的 `baseURL`：
```javascript
const request = axios.create({
  baseURL: 'http://your-api-url/BookManager',
  timeout: 10000
})
```
